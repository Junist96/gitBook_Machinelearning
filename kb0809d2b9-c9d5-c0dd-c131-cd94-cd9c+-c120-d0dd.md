# 특징 생성 

특징 생성 = 특징 추출 + 특징 선택 

특징생성 정의: 외부 세하는 패턴으로부터 특징 벡터를 만드는 전체 과정 계에 존재

특징은 두 가지 기준에서 우수 해야 한다. 
- 분별력(Discriminatory): 서로 다른 부류를 잘 분별해 주어야 한다. 
- 차원(Demensionality): 차원이 낮을수록 계산 효율이 좋다.
