# 확률 분포 추정 

베이즈 정리 = $$P(\omega_i \mid X ) = \frac{p(X\mid \omega_i)P(\omega_i}{p(X)} = \frac{우도 \times 사전확률}{p(X)}$$

- 베이즈 정리를 완성 하려면 `우도` 값을 알아야함
- 우도값은 데이터의 분포를 가정할수 있다면 쉽게 가능 
    - 쉬운 이유 : 분포의 매개 변수만 알면 가능
